# Chat with @tachikoma

The @tachikoma chat participant routes your prompts to the Hermes runtime
on your computer and surfaces MCP tool calls inline.

- [Open chat with @tachikoma](command:workbench.action.chat.open?%7B%22query%22%3A%22%40tachikoma%20%22%7D)
- Mention agents directly: `@tachikoma plan a refactor of ...`
- Replies stream live with tool output and file references.
