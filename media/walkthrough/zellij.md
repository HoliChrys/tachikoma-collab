# Open a zellij terminal

Click the dropdown next to the New Terminal button and pick
**Tachikoma (zellij)** to attach to the current context's shared
zellij session through zweb.

- Sessions are persistent and shared across collaborators.
- Drop your active context first so the terminal lands in the right space.
- Use [Open Remote Terminal](command:tachikoma.remoteTerminal) for a plain
  SSH shell instead.
