# Connect with API Token

Paste a Tachikoma API token issued by the CLI or dashboard. The editor will
verify the token, store it in VS Code SecretStorage, and reconnect
automatically across reloads.

- Run: [Tachikoma: Connect with API Token](command:tachikoma.connectWithToken)
- Tokens last 24h and rotate in the background.
- Status appears in the bottom status bar.
