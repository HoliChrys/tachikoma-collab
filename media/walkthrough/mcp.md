# Manage MCP profile

Each MCP profile scopes which tools your agents can use. Pick one to
match the task you are working on.

- [Open the MCP Copilot view](command:tachikoma.copilot.open)
- [Select an MCP profile](command:tachikoma.mcp.selectProfile)
- [Clear active profile (union mode)](command:tachikoma.mcp.clearActiveProfile)
- The lightning bolt on the status bar means an MCP session is live.
